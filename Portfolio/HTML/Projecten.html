Projecten

In de tijd dat ik op deze school heb gezeten heb ik aan veel projecten meegedaan.
Hieronder zijn links van de verschillende projecten. En een beschrijving van elk project.

Project JS: Rekenmachine

Bij dit project moest ik een werkende rekenmachine maken met behulp van javascript HTML en CSS en dan presenteren. Wat best goed ging. Waar ik vooral trost om was is dat ik een echt werkende rekenmachine heb kunnen maken.

Project JS: Boter, kaas en eieren

Bij dit project moest ik het spel boter, kaas en eieren maken met behulp van javascript HTML en CSS. Als je naar beneden scrolt kan je het spel ook zien.

Project C#: Dice game

In dit project moest ik met 3 andere studenten een dice game maken met behulp van C#. Wat best leuk maar vooral leerzaam was.

Project C#: Dierentuin

Dit was mijn eerste project op school. Ik moest samen met 3 andere studenten er voor zorgen dat we met behulp van C# een for-loop, switch en if-statement kunnen herkennen, maken en toepassen. En we moesten ook versiebeheer kunnen toepassen (Git).

Project C#: Higher lower

Bij het project hoger en lager moest ik met 4 andere leerlingen een spel maken. We moesten de behoefte van de product owner in kaart brengen, Het hoger-lager spel ontwerpen en een tussenpresentatie maken. Daarbij moesten we ook een eindverslag maken.

Beroepsprofiel

Ik moest ook een beroepsprofiel maken. De opdracht was als volgt: Je zoekt een bedrijf waar je graag zou willen werken, en interview iemand die daar werkt of soortgelijk werkt doet. Waar ik graag wou werken was Discord. Dus heb ik samen met een medestudent een gesprek proberen te regelen met een programmeur en hem een korte interview gegeven. Hij vertelde waar zijn bedrijf voor staat, wat hij leuk vond aan zijn werk, wat zijn taken waren als programmeur en wat nodig was om een geschikte programmeur te worden.
